Admin: REDCap Data & Provider Dashboards

Study team use only. Upload a REDCap CSV export to generate personalized dashboard links for each provider.

🗺 How the Registry Connects

The three parts of this site are intentionally separate — registration, dashboards, and the patient enrollment page are independent workflows:

📝 Provider Registration

MDs fill out the form on the Provider Portal. Submissions are saved to a private Google Sheet and emailed to the study team. Providers automatically receive a confirmation email with their dashboard link.

📊 Provider Dashboards

Separate from registration. Upload a REDCap CSV here → generate a unique link per provider → email each provider their link. No login needed.

📱 Patient Enrollment

The QR code on the Provider Portal links patients to enroll.html. Patients click through to the REDCap consent survey. No connection to registration.

🔄 REDCap Data Updates

REDCap doesn't connect directly to this site. Export a fresh CSV weekly (or as needed) and re-upload here to generate updated dashboard links.

⚙ One-Time Setup — Google Apps Script
ℹ
Provider registrations are stored in a private Google Sheet in your Stanford Drive and emailed to the study team. Providers automatically receive a confirmation email with their dashboard link. Do this once before going live.
  1. Open script.google.com → New project → paste the entire contents of google-apps-script.js (included in the site folder), replacing any existing code → Save
  2. Click Project Settings (gear icon) → Script Properties → Add property: name ADMIN_KEY, value = a long random password you choose (e.g. SURN-Admin-2026-XK9) → Save
  3. Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy → copy the Web App URL
  4. Open assets/js/app.js and replace PASTE_YOUR_WEB_APP_URL_HERE with the URL you just copied
  5. Enter that same Web App URL and Admin Key below to view registered providers from this admin page

🤖 reCAPTCHA Setup
  1. Go to google.com/recaptcha → reCAPTCHA v2 → “I’m not a robot” checkbox → add your site's domain
  2. Copy the Site Key (public) and open index.html
  3. Replace YOUR_RECAPTCHA_SITE_KEY in the data-sitekey attribute with your site key
  4. Copy the Secret Key and add it to your Formspree settings under “reCAPTCHA” to enable server-side validation
